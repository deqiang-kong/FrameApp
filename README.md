# 简介

该项目是一个学习项目，目的是将当前比较成熟流行的技术点运用到项目中通过动手实践将该踩的坑踩一遍。




# 界面功能


### 知乎日报

### 网易新闻

### 视频




# 技术点
### 整体开发模式
- MVP

### 主体采用kotlin

### 依赖注入

### 网络请求及异步处理：
- RxJava2+Retrofit2
- Okhttp
- Gson
- RxBus 实现事件总线功能

### 图片加载：
- glide

### 视频播放器 ：
- JieCaoPalayer

### 内存泄漏检测 ：
- leakcanary

### 其他：
- 图片毛玻璃效果（头像背景效果）
- 自定义封装的 RecyclerView： [MagicRecyclerView][7] 支持头部底部和标签
- 自定义 behavior 实现滑动时底部导航栏的隐藏显示效果
- Android N 文件读写适配
- Android M+ 运行时权限申请封装
- 分享 ShareSDK 接入（分享平台未进行注册申请，后续再完善）
- 自定义相册图片选择
- Activity 右滑返回
